## Featured Projects

### 📊 [Funnel Analysis: E-commerce Conversion Optimization](https://github.com/Aijarkynhref/funnel-analysis-trendyol)
**Problem:** Fashion category conversion dropped 15%. Where are customers dropping off?  
**Solution:** Built SQL-based funnel analysis (Views → Cart → Checkout → Purchase). Found 50% drop at Add-to-Cart stage.  
**Impact:** Recommended A/B test (improved product photos + pricing) → projected 20% conversion uplift.  
**Tech:** SQL, Python (pandas, seaborn), Jupyter

---

### 📈 [Demand Forecasting: Inventory Optimization](https://github.com/Aijarkynhref/time-series-forecasting)
**Problem:** E-commerce seller facing stockouts during peak seasons + overstock on slow-movers.  
**Solution:** Time-series forecasting model using Prophet to predict demand 3 months ahead.  
**Impact:** Enabled data-driven procurement decisions, reducing stockouts by 30%.  
**Tech:** Python (Prophet, pandas, statsmodels), Jupyter
